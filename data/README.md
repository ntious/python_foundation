# Data Folder

This folder contains small practice datasets for **IT1009C: Introduction to Python for Information Technology**.

The files are intentionally simple. They are designed for absolute beginners who are learning how to read CSV files, store values in variables and lists, use loops, compare values, and write basic `if` statements.

These files are for classroom practice only. They are not real student records, financial records, or production login data.

## Files

### `sample_scores.csv`
Sample student names and scores.

Students can use this file to:
- calculate an average score
- find the highest or lowest score
- count scores above or below a passing mark
- practice simple pass/fail rules

### `sample_expenses.csv`
Sample everyday expenses with an item, category, and amount.

Students can use this file to:
- calculate total spending
- add spending by category
- find expenses above a chosen amount
- practice basic budgeting decisions

### `sample_logins.csv`
Sample login results with a username, login status, and number of attempts.

Students can use this file to:
- count successful and failed logins
- find users with repeated failed attempts
- compare success and failure counts
- practice simple IT security rules

### `sample_data_dictionary.md`
A short guide that explains each column in plain language.

## Teaching Notes

- The datasets are small on purpose.
- Column names use clear `snake_case` names.
- Values are clean and consistent so students can focus on Python basics.
- Instructors may ask students to edit or expand the files during practice activities.
